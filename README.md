# 智协CampusHub - 校园互助服务平台

## 项目简介

“智协CampusHub”是一个针对校园场景开发的互助服务平台，旨在为在校学生提供便捷的互助交流社区。平台支持学生发布和承接各类互助需求（涵盖快递代取、学习辅导、二手交易、活动组队等）。同一个用户可以在“需求方”和“服务方”角色之间灵活切换。平台提供基于分类、地理位置、时间的智能匹配，包含完整的订单状态流转（接单-确认-进行中-完成-评价），并引入双向评价与信用分体系，以保障交易的安全性与可靠性。

## 技术栈

**前端技术栈**

- 核心框架：Vue
- UI 组件库：Element Plus
- 状态管理：Pinia
- 网络请求：Axios

**后端技术栈**

- 核心框架：SpringBoot
- 数据持久层：MyBatis
- 安全鉴权：JWT (JSON Web Token)

**数据库与中间件**

- 主数据库：PostgreSQL 15+
- 空间数据处理：PostGIS (用于地理围栏和距离筛选)
- 缓存中间件：Redis
