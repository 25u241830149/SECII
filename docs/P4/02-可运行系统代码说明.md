## 7. 启动方式

启动方式以 README 末尾“快速启动指令”为准。

### 7.1 启动数据库和 Redis

```powershell
docker compose up -d
```

当前 `docker-compose.yml` 会启动：

- `postgis/postgis:16-3.5`，数据库 `secii_db`
- `redis:7-alpine`

如果不用 Docker，需要手工准备 PostgreSQL/PostGIS 与 Redis，并保证 `application-dev.yml` 中的连接信息可用。

### 7.2 启动后端

```powershell
cd backend/campushub-parent
mvn -pl campushub-bootstrap -am -DskipTests package
java -jar campushub-bootstrap/target/campushub-bootstrap-0.0.1-SNAPSHOT.jar
```

默认地址：

- 后端：`http://localhost:8080`
- Swagger：`http://localhost:8080/swagger-ui.html`

说明：`application-dev.yml` 当前配置 `spring.sql.init.mode=always`，后端启动时会执行幂等 schema 初始化。

### 7.3 启动前端

```powershell
cd frontend
npm.cmd install
npm.cmd run dev -- --host 127.0.0.1
```

默认地址：

- 前端：`http://127.0.0.1:5173/`
