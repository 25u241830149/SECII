# “智协”校园互助平台调研方案

## 一、调研目标
本次调研旨在深入了解大学生在校园生活中的真实互助需求，明确用户对校园互助平台的功能期望、信任机制、收费偏好及使用习惯，为“智协”校园互助平台的功能设计、优先级排序及用户体验优化提供真实数据支撑。

## 二、调研方法
采用定量问卷+定性访谈相结合的方式：
- **问卷调研**：线上发放结构化问卷，收集用户对平台各模块的偏好、注册方式态度、收费意愿等量化数据。
- **用户访谈**：半结构化访谈，挖掘问卷无法覆盖的深层需求、使用痛点和改进建议。

## 三、问卷设计
问卷由课程统一设计，共 13 题，具体内容如下：
1. 给出你希望有的板块
2. 请问你认为互助平台应该包含哪些功能？
3. 以上功能或板块中，你认为最重要的是？
4. 你认为通过学号认证注册方式如何？
5. 为了让平台活下去且维持良好氛围，你认为需要管理哪些方面？
6. 你希望平台的收费方式是？
7. 如果平台需要志愿管理员，你的应聘意愿是？
8. 你对平台建设还有哪些建议？
9. 要是平台真建成了且真投入使用了，你的使用意愿是？（1-10分，10为意愿极高）
10. 你在生活中还有什么需要他人参与协助的需求？
11. 你在互助平台上想要得到怎样的酬劳，你愿意接受本平台的积分吗？
12. 在互助的过程中，你认为需要与对方进行线上交流吗？
13. 你认为平台应该通过什么标准来判断一个互助任务成功或者失败？

## 四、访谈提纲
1. 你目前在微信群、QQ群等现有渠道寻求校园互助时，遇到的最大困难是什么？比如是否存在求助无人回应、找不到合适的协助者、沟通效率低等问题，请结合具体场景说明。
2. 如果有一个专门的CampusHub校园互助平台，结合你日常校园生活，你最希望它帮你解决哪一类具体互助问题？请举例说明这类问题的高频程度和迫切性。
3. 针对校园常见的跑腿类互助，你愿意支付多少报酬？优先选择现金还是平台积分？请说明你确定这个金额/偏好的原因（如成本、便捷性等）。
4. 若在互助过程中，遇到对方不履约或态度恶劣的情况，你希望平台采取哪些具体处理措施？比如是否需要投诉通道、信用惩罚、赔偿机制等。
5. 你希望 CampusHub 平台增加匿名发帖功能吗？如果希望，具体会在哪些校园互助场景下使用；如果不希望，请说明理由。
6. 对于平台拟推出的信用分体系，你认为信用分应从哪些具体维度评定？信用分高低应对应哪些不同的权益或限制？
